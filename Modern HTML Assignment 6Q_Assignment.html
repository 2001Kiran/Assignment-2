5 semantic tags in HTML:

Semantic HTML tags provide meaning to the content they encapsulate, making it easier for both humans and machines to understand the structure and purpose of a web page. Here are five commonly used semantic tags in HTML along with their descriptions:

  1. < header >:

    Description: The < header > element represents introductory content or a group of navigational links typically found at the top of a webpage or section. It often includes the site's logo, main navigation menu, search form, or other introductory elements.

    Example Usage: < header >< h1 >My Website< /h1 >< nav >...< /nav >< /header >

  2. < nav >:

    Description: The < nav > element is used to define a section of a webpage that contains navigation links allowing users to navigate within the website or to other related pages. It typically includes menus, lists of links, or other navigation elements.

    Example Usage: < nav >< ul >< li >< a href="/">Home< /a >< /li >< li >< a href="/about">About< /a >< /li >< /ul >< /nav >

  3. < main >:

    Description: The
    element represents the main content area of a webpage, excluding any header, footer, or sidebar content. It should contain the primary content that is directly related to the purpose of the webpage.

    Example Usage: < main >< article >< h2 >Article Title< /h2 >< p >Article content...< /p >< /article >< /main >

  4. < article >:

    Description: The < article > element is used to define a self-contained piece of content that can be distributed and reused independently. It typically represents a blog post, forum post, news article, or any other independent content item.

    Example Usage: < article > < h2 > Blog Post Title< /h2 > < p >Blog post content...< /p > < /article >

  5. < footer >:

    Description: The < footer > element represents a footer section typically found at the bottom of a webpage or section. It often includes copyright information, contact details, links to related pages, or other supplementary information.

    Example Usage: < footer >< p > & copy; 2024 My Website< /p > < nav >...< /nav > < /footer>

Using semantic HTML tags not only improves the accessibility and SEO-friendliness of a webpage but also makes it easier to maintain and understand the structure of the document.
